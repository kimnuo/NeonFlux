  1. Android 빌드 문제 해결
────────────────────────────────────────────────────────────────────────────────

  [문제점]
  - Gradle 빌드 실패: CommandInvokationFailure
  - Kotlin 라이브러리 버전 충돌 (kotlin-stdlib 2.1.0 dexing 실패)
  - kotlinx-coroutines 1.8.0 dexing 실패
  - AndroidResolverDependencies.xml에서 ABI가 armeabi-v7a만 설정됨
  - AndroidTargetSdkVersion이 0으로 미설정
  - gradleTemplate.properties에 SDK 36 호환성 코드 존재

  [수정 내용]
  - ProjectSettings/ProjectSettings.asset
      * AndroidTargetArchitectures: 1 → 3 (ARM64 + ARMv7)
      * AndroidTargetSdkVersion: 0 → 34
      * useCustomMainGradleTemplate: 0 → 1
      * useCustomGradleSettingsTemplate: 0 → 1
      * applicationIdentifier: com.UnityTechnologies... → com.kmu.neonflux

  - ProjectSettings/AndroidResolverDependencies.xml
      * androidAbis: "armeabi-v7a" → "arm64-v8a,armeabi-v7a"
      * Google Ads 의존성 패키지 전체 제거

  - Assets/Plugins/Android/gradleTemplate.properties
      * android.suppressUnsupportedCompileSdk=36 제거
      * android.jetifier.ignorelist 제거

  ■ 데이터 흐름 다이어그램
  ─────────────────────────────────────────────────
  게임 진행 중:
    PlayerController (점수 계산)
      → UpdateScoreText()
        → GameManager.SetCurrentScore(score)  [실시간 업데이트]

  게임 종료 시 (장애물 충돌 / 코스 이탈):
    GameManager.EndGame() 또는 SetGameOver(reason)
      → RecordGameScore(reason)
        → SaveManager.RecordGameScore(score, reason)
          1. CurrentData.highScore 갱신 (현재 점수 > 기존 최고 점수)
          2. LeaderboardEntry 생성 (score, stageLevel, date, reason)
          3. scoreHistory.Add(entry)
          4. 10개 초과 시 오래된 항목 제거
          5. File.WriteAllText()로 디스크 저장
      → ApplyStateVisuals()
        → GameOverUI.ShowGameOver(score, reason)
          - 현재 점수 표시 (Neon Cyan)
          - 이전 점수 표시 (Neon Magenta)
          - 최고 점수 표시 (Neon Yellow / Green if NEW)
          - NEW HIGH SCORE! 효과
          - 리더보드 상위 3개 요약

  게임 클리어 시:
    GameManager.CompleteStage()
      → RecordGameScore("clear")
      → SaveManager.RecordGameScore(score, "clear")
      → currentStageLevel++
      → SaveData()

  메인 메뉴:
    GameManager.GoToMainMenu()
      → mainMenuUI.gameObject.SetActive(true)
      → MainMenuUI.OnEnable()
        → UpdateHighScoreDisplay()
        → ShowMainPanel()

  리더보드 보기:
    MainMenuUI.OnLeaderboardClicked()
      → ShowLeaderboardPanel()
      → LeaderboardUI.RefreshLeaderboard()
        → SaveManager.Leaderboard (정렬된 목록)
        → 상위 순위 표시 (Gold/Silver/Bronze)
